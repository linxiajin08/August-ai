# 语音助手项目

## 安装依赖

1. 创建虚拟环境（推荐）：
```
python -m venv venv
```

2. 激活虚拟环境：
- Windows: 
```
venv\Scripts\activate
```
- Mac/Linux: 
```
source venv/bin/activate
```

3. 安装依赖库：
```
pip install -r requirements.txt
```

## 运行项目

1. 确保虚拟环境已激活
2. 运行主程序：
```
python ui_app.py
```

## 注意事项
- 需要麦克风设备支持语音输入
- 首次使用需要配置API URL和密钥
